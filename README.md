# CarPI

## Requirements
* pip
```
$ sudo apt-get install python3-pip
```

* OpenCV 3.6
```
$ sudo pip3 install opencv-python
```

* I2C enabled on raspberry pi
* Install that packages:
```
$ sudo apt-get install -y python-smbus
$ sudo apt-get install -y i2c-tools
```

## Target environemnt

* Start up
```
$ python3 main.py
```
